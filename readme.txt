== Description ==

Amazon Search Utility for any site page. Allows your visitors to search all of Amazon for any item they desire. Your Associate Account is Credited with any purchase they may make.

= Benefits =

Increase your Amazon Affiliate Earnings. Search and Sales capability for your visitors with your Associate Tag Credited for any sale.

= Feature List =

* Searches Amazon and Displays an Amazon Products Listing on Your Page,
* Your Amazon Associate Account is Credited for all Sales,
* Select Desired Country (Professional Version has many more),
* Select Desired Category (Professional Version has many more),
* Keep Adding More Sub Categories for a More Refined Search,
* Enter Search For Parameters to Narrow in on Exact Products,
* Displays 10 products in Listing (Professional Version shows 100),
* Two Screen Modes (Normal and Wide) to Fit Your Specific Theme

= Example usage =

**Admin** - Enter Your Amazon Associate information in the Admin Screen. Place the Short Code '[Amazon_Explorer]' on any screen. Amazon Explorer will then appear where the Short Code has been placed when that Screen is Displayed.

**User** - Use the Category Selector to define any search by Country, Categories and many Sub Categories and Item Specifics. Results displayed immediately below for all searches and Full detail pages are available for Display for each product by clicking View or Double Clicking any Product Line Displayed.

== Installation ==

1. Upload the FOLDER 'Amazon Explorer' to /wp-content/plugins/.
2. Activate the plugin 'Amazon Explorer' through the 'Plugins' menu in admin.
3. Go to 'Settings' and then 'Amazon Explorer' to Set Parameters.
